# 如何高效地实习

我想先表达一个核心观点：**我们实习、工作，从来不只是为了公司，更是为了自己。**

打工的意义，不应该只停留在“完成眼前的工作”，而应该是通过每一段经历，持续积累能力、认知和经验，逐步摆脱被动打工的处境。也许这在现在的环境来看并不容易，但我们要把它当作一颗种子，埋在心里慢慢培育它。

只有带着这样的心态，你才会真正拥有自我驱动力。

## 一、先搞清楚自己所在的业务

进入一个部门后，第一件事不是急着埋头做需求，而是先理解你所在的业务环境。你至少需要想清楚下面几个问题：

- 部门的核心业务是什么？
- 产品是为谁服务的？
- 解决了哪些真实问题或痛点？
- 当前产品处于什么阶段？
- 目前接到的需求，主要在解决什么问题？

## 二、日常工作中养成两个重要习惯

### 1. 不会很正常，但要学会高质量提问

对于实习生来说，很多事情不会是非常正常的。刚开始时，你可以有很多问题，也应该大胆去问。

前期可以是大量问题，随着你对业务和技术逐渐熟悉，后续要慢慢沉淀成更少但更精准的问题。也就是说，你要从“遇到什么都问”，逐步过渡到“先自己思考、再带着判断去问”。

这不仅能提升沟通效率，也能让别人更愿意帮助你。

### 2. 有任务意识，也要有协作意识

接到任务时，不要立刻埋头开始做，而是先确认几件事情：

- 任务的优先级是什么；
- 截止时间（DDL）是什么；
- 是否有明确的交付标准；
- 有没有潜在风险需要提前说明。

在此基础上，你可以主动给出自己的预计完成时间。这个时间最好给自己预留一定 buffer，不要把计划排得过满。一旦发现中途有风险，或者进度可能受影响，要尽早暴露，而不是等到最后一刻才说做不完。

同时，要养成**主动同步进度**的习惯。不要长时间闷头做事、没有反馈。尤其是实习阶段，沟通本身就是工作的一部分。比较好的方式是：每半天或每天固定同步一次当前的todo、进展，让 mentor 或同事能及时了解你的状态，也方便他们在必要时提供帮助。
